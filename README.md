# :cloud: cloudOS :cloud:

Learning about operating systems by creating my own operating system.

## Development

To build:

```bash
cargo xbuild
```

To build and run in QEMU:

```bash
cargo xrun
```

To run unit tests:

```bash
cargo xtest
```
